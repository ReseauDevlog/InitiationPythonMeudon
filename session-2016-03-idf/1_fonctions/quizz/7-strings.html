On veut écrire une fonction affichage qui met en forme un message à partir de 3 variables (nom, age, no_secu).
Quelle(s) solution(s) est(sont) correcte(s) ?(plusieurs réponses possibles)

		
Pour obternir par exemple :


>>> affichage("Dupont", 25, "1975787")
'Dupont, 25 ans, 1975787'

>>> affichage("Durand", 17, None)
'Durand, 17 ans, pas de no de SS'

PS : Remarquez bien que la fonction affichage doit retourner le message en valeur, et non pas réaliser l'impression elle-même.


On envisage plusieurs solutions pour réaliser cela :



Solution 1 :


def affichage(nom, age, no_secu):
    return "{nom}, {age} ans, {secu}".\
          format(nom=nom, age=age, 
                 secu = no_secu if no_secu else "pas de no de SS")


Solution 2 :


def affichage(nom, age, no_secu):
    return "{}, {} ans, {}".\
          format(nom, age, no_secu if no_secu else "pas de no de SS")


Solution 3 :


def affichage(nom, age, no_secu):
    return "%s, %s ans, %s".\
          format(nom, age, no_secu if no_secu else "pas de no de SS")


Solution 4 :


def affichage(nom, age, no_secu):
    print nom + ", " + age + " ans, " +\
          (no_secu if no_secu else "pas de no de SS")


 

		

			   
			
		

		

			   
			
		

		

			   
			
		

		

			   
			
		

		

			Afficher la solution
		

		

			Les clauses 1, 2 sont correctes


La clause 3 est erronée, elle mélange les %s de l'opérateur % avec la méthode format


La clause 4 est erronée car elle fait print au lieu de return. Le code qui élabore le message serait par ailleurs correct, mais paraît moins lisible que les options 1 et 2.
